# Arbeit mit GitHub

Das Projekt ist folgendem GitHub Repository vorhanden: https://github.com/mio42-GmbH/Prototyp-MIO-Anzeige-Impfpass-JSSPA

## Github Account Erstellen

Um an diesem Projekt zu mitzuentwickeln, wird ein GitHub Account benötigt.
Dieser kann einfach auf https://github.com/ erstellt werden, unter "Sign up" oben rechts.

<img src="./githubSignUp.png" alt="Bei GitHub über 'Sign Up' registieren" />

## Lokales Git Installieren

Weiterhin muss Git auf dem lokalen Rechner installiert sein.
Dazu empfehlen wir die Installation eines grafischen Git-Clients wie GitHub-Desktop: https://desktop.github.com/
Dieser Client von GitHub installiert eine gebündelte Version von Git auf dem System.

Alternativ kann mit anderen Clients wie SourceTree (https://www.sourcetreeapp.com/), der integrierten Git-Funktionalität in VS Code, oder ohne grafische Oberfläche auf dem Terminal gearbeitet werden.
In diesen Fällen muss eine separate Git-Installation auf dem Betriebssystem erfolgen.
Für Windows empfiehlt sich Git for Windows: https://git-scm.com/download/win

Für Mac OSX, empfehlen wir die Installation von Git per Homebrew mit

<code>brew install git</code>

Für Linux empfiehlt sich die Installation per Standard Package Manager, für Ubuntu beispielsweise

<code>sudo apt-get install git</code>

## Branches

### Branch erstellen

Da mehrere Entwickler an diesem Projekt gleichzeitig arbeiten, empfiehlt es sich, nicht direkt auf dem master-Branch zu arbeiten.

Ein neuer Branch sollte bei der Arbeit an neuen Features/Fixes erstellt werden und einen sinnvollen sprechenden Namen erhalten.

#### Branch erstellen in GitHub Desktop

<img
  src="./githubDesktopCreateBranch.png"
  alt="Branch in GitHub Desktop erstellen"
/>

#### Branch erstellen in VS Code

Aktueller Branch wird unten links in Statusleiste angezeigt, zum Branch wechseln anklicken:

<img
  src="./vsCodeBranchButton.png"
  alt="Knopf um Branch in VS Code zu wechseln"
/>

Anschließend "Neuen Branch erstellen" wählen und einen Branchnamen eingeben

<img src="./vsCodeCreateBranch.png" alt="Neuen Branch in VS Code erstellen" />

Neuer Branchname sollte in Statusleiste angezeigt werden

<img
  src="./vsCodePublishBranchButton.png"
  alt="Neuen Branch in VS Code veröffentlichen"
/>

### Branches Mergen

Nachdem die Arbeit auf dem Branch abgeschlossen wurde und ein funktionierender Stand hergestellt wurde, kann der Branch wieder in den Master-Branch gemerged werden.

Dazu müssen alle Änderungen committed werden.

Anschließend muss jeweils der Master-Branch, in den gemerged werden soll, wieder ausgewählt werden.

#### Branches Mergen in GitHub Desktop

<img
  src="./githubDesktopMergeBranch.png"
  alt="Branches in GitHub Desktop mergen"
/>

#### Branches Mergen in VS Code

In VSCode können Branches über das Kontextmenü in der Quellcodeverwaltung gemerged werden:

<img src="./vsCodeBranchMenu.png" alt="Branch Kontextmenü in VS Code" />

anschließend "Branch" > "Branch zusammenführen" auswählen
